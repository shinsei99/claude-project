# 媒介契約書ジェネレーター

不動産の**謄本（登記事項証明書）PDF を最大5枚アップロード**すると、
**土地 / 建物 / マンションを自動判別**して内容を読み取り、
**媒介契約書（一般・専任・専属専任）の Excel** を自動作成する業務支援アプリです。

国土交通省の標準媒介契約約款に基づくフォーマット（`templates/original_baikai_template.xls`）の
主要部を再構成し、別表（目的物件の表示）に謄本の内容を自動転記します。

## 特長

- 📎 **謄本を最大5枚同時アップロード**（土地謄本＋建物謄本＋区分建物を混在OK）
- 🤖 **土地/建物/マンションを自動判別**（土地＋建物 → 自動で「土地建物」に統合）
- ✍️ **AI読み取り**（ローカルの `claude` CLI を呼び出し。APIキー不要）
- 🧭 **スキャン謄本の向きを自動補正してから解析**：横向き・逆さスキャンも haiku で回転角（0/90/180/270）を判定し、正立画像に直してから読み取るので精度が安定
- 🧾 **別表を自動入力**：所有者・登記名義人・物件所在地・地番/地目/地積・種類/構造/床面積/新築年月日・専有面積/敷地権 等
- 💴 **約定報酬を速算式で自動計算**（価格×3%＋6万円・消費税10%）
- 📄 **3種別に対応**：専任 / 専属専任 / 一般（条項・約款・報告頻度・流通機構登録期間を種別ごとに反映）
- 📚 出力 Excel は「媒介契約書」＋「約款」の2シート構成

## 仕組み

```
謄本PDF(最大5枚)
  ├─ テキスト層あり ──pdfplumber──▶ テキスト ─────────────┐
  └─ テキスト層なし(スキャン) ──PyMuPDFで画像化──▶ haikuで向き判定→正立補正 ─┤
                                                          ▼
                                          claude CLI(sonnet)で構造化JSON抽出
                                                          │ 土地/建物/マンションを統合・マージ
                                                          ▼
   入力フォーム(甲・期間・媒介価格・特約) ＋ 別表自動入力 ──openpyxl──▶ 媒介契約書.xlsx
```

- スキャンPDFは「①各ページ画像化 → ②向き自動補正（haiku）→ ③AI読み取り（sonnet）」の順で処理します。
- CLI が使えない環境では正規表現フォールバックで主要項目のみ抽出します（要確認の警告を表示）。
- 完全自動ではなく「下書き生成・転記支援」が目的です。出力は必ず内容を確認してください。

## ディレクトリ構成

```
baikai-generator/
├── app.py                         # Streamlit UI
├── services/
│   ├── registry_parser.py         # 謄本PDF → 構造化データ（pdfplumber + claude CLI）
│   ├── excel_builder.py           # 媒介契約書 xlsx 生成（別表＋約款）
│   └── contract_text.py           # 約款の正規テキスト（元xlsから抽出）
├── templates/
│   └── original_baikai_template.xls  # 元フォーマット（参照用）
├── smoke_test.py                  # 依存なしで通る軽量テスト
├── requirements.txt
└── README.md
```

## セットアップ・起動

```bash
cd baikai-generator
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py --server.port 8514
```

ブラウザで http://localhost:8514 を開きます。
AI読み取りにはローカルに `claude` CLI（Claude Code）が必要です（`~/.local/bin/claude`）。

## テスト

```bash
python smoke_test.py
```

物件種別の統合ロジック・約款データ・Excel生成（一般/専任/専属専任/マンション）を検証します。

## 使い方

1. サイドバーに自社（乙）の情報を入力（任意・保存はされません）
2. 謄本 PDF を最大5枚アップロードし「謄本を解析して読み取る」
3. 判別された物件種別と読み取り結果を確認・補正
4. 依頼者（甲）・契約日・有効期間・媒介価格・特約を入力
5. 契約の種類（専任/専属専任/一般）を選び「Excelを生成」→ ダウンロード

## 注意

- 媒介価格・報酬・有効期間など金額や日付に関わる項目は、生成後に必ずご確認ください。
- 約款は標準媒介契約約款に基づく内容ですが、実務では自社の最新書式・特約をご確認のうえご利用ください。
