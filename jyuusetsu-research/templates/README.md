# templates/ — 書式テンプレートの置き場

**個人情報保護のため、実テンプレート(.xlsx)は Git にコミットしていません**
（このリポジトリは公開のため）。別PCで作業する場合は、以下のファイル名で
各書式の Excel テンプレートをこのフォルダに配置してください。
セル位置のマッピングは `services/format_export_service.py` の `FORMATS` を参照。

| ファイル名 | 書式 | シート名 |
|------------|------|----------|
| `rental_building_template.xlsx` | 賃貸重説（建物賃貸借用 A4） | `建物賃貸借用` |
| `sale_landbuilding_template.xlsx` | 売買契約書（土地建物・公募用 一般売主） | `(8)土地建物公薄用売買契約書（一般売主）` |
| `sale_mansion_contract_template.xlsx` | 売買契約書（区分所有建物・敷地権 宅建業者売主） | `(13)区分所有建物用（敷地権）売買契約書(宅建業者売主)` |
| `sale_mansion_jyuusetsu_template.xlsx` | 重要事項説明書（区分所有建物の売買・交換用） | `(2)重要事項説明書(区分所有建物の売買・交換用)` |

- 汎用ドラフト用 `jyuusetsu_template.xlsx` はアプリ起動時に自動生成されます（配置不要）。
- テンプレートは個人情報を含まない**白紙版**を用いることを推奨します。
- 書式を追加するときは、ここにファイルを置き `FORMATS` に1エントリ追加します。
